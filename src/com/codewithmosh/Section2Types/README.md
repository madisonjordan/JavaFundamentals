### primitive types
|  Type   | Bytes |    Range     |
|:-------:|:-----:|:------------:|
|  byte   |   1   | [-128, 127]  |
|  short  |   2   | [-32K, 32K]  |
|   int   |   4   |  [-2B,2B ]   |
|  long   |   8   |              |
|  float  |   4   |              |
| double  |   8   |              |
|  char   |   2   |   A, B, C    |
| boolean |   1   | true / false |

### reference types

anything that is not in the list of primitive types
